# Hunar-e-Hajar (Portfolio Website)

## Run locally
- If you have Python installed:
  - `python -m http.server 5173`
  - Open `http://localhost:5173/`
- Or just double-click `index.html` to open it directly (some browsers may restrict module loading from file URLs).

## Add paintings
Place the painting images in `assets/paintings/` and name them to match `site/data.js`:
- `still-life.jpg`
- `landscape.jpg`
- `koi.jpg`
- `forest.jpg`
- `heritage-fort.jpg`
- `heritage-domes.jpg`
- `heritage-shrine.jpg`
- `floral-blooming-set.jpg`
- `horse.jpg`

You can change titles, descriptions, categories, and filenames in [data.js](file:///d:/TRAE%20Projects/site/data.js).

## Update Instagram link
Edit `instagramUrl` in [data.js](file:///d:/TRAE%20Projects/site/data.js).
